Detailed Description

Base exception class for other exceptions to inherit from.

The parser uses this exception class to polymorphise other related exceptions such as InvalidTransactionSizeException, InvalidBlockSizeException, and more. Anytime such an exception is thrown, execution should be stopped either because file could not be read or the blockchain being read is invalid.
